# 📊 Sistema de Gestão de Promissórias (SGP)

O **SGP** é uma aplicação robusta e leve desenvolvida para a gestão completa de títulos de crédito e automação de processos financeiros. O grande diferencial deste projeto é a sua arquitetura **Standalone**: todo o código (HTML, CSS e JavaScript) está integrado, facilitando a execução e portabilidade sem a necessidade de instalações complexas ou servidores de backend.

---

## 🌟 Diferenciais do Projeto

- **Arquitetura All-in-One:** CSS e JavaScript embutidos no HTML para máxima portabilidade.
- **Zero Dependências:** Não requer Node.js, bancos de dados SQL ou servidores; roda diretamente no navegador.
- **Armazenamento Inteligente:** Persistência de dados via arquivos **JSON**, permitindo que o usuário escolha a pasta de salvamento para maior controle e backup.

---

## 🛠️ Funcionalidades

### 1. 👥 Cadastro de Clientes
- Registro completo de informações de contato.
- Organização centralizada para consulta imediata de histórico.

### 2. 🛒 Tela de Vendas
- Interface intuitiva para lançamento de novas transações.
- Cálculo automático de parcelas e geração dos títulos de promissória.

### 3. 🏦 Acompanhamento Financeiro & CRM
- **Gestão de Cobrança:** Módulo de CRM focado no acompanhamento de status (Em aberto, Negociação, Atrasado).
- **Controle de Fluxo:** Visualização clara de liquidações e pendências financeiras.

### 4. 📈 Relatórios Gerenciais
- Dashboards integrados para análise de performance.
- Relatórios de saúde financeira para suporte à tomada de decisão.

---

## 🚀 Como Executar

Por ser um sistema baseado em tecnologias front-end puras e embutidas, o uso é extremamente simples:

1. **Baixe o arquivo do sistema** (ex: `index.html`).
2. **Abra com qualquer navegador moderno** (Chrome, Edge, Firefox).
3. **Configure a pasta de dados:** No primeiro acesso, defina o diretório onde o arquivo JSON de banco de dados será manipulado.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica.
- **CSS3:** Estilização moderna e responsiva (Internal Stylesheet).
- **JavaScript (Vanilla):** Lógica de negócio, manipulação de DOM e gestão de arquivos JSON (Internal Scripts).
